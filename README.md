# FreeStack App

Monorepo fullstack com estrutura organizada para backend (Node.js/Express) e frontend (React).

## Estrutura

```
freestack-app/
├── backend/          # API Express
├── frontend/         # Frontend React
├── .github/workflows # CI/CD
├── docker/           # Docker config
```

## Scripts

```bash
cd backend
npm install
npm run dev

cd ../frontend
npm install
npm start
```
